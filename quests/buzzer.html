Quest: Passive Buzzer

In this quest, you‚Äôll learn how to use a passive buzzer with your Arduino to generate sound using the tone() function.

Required Components

  • Arduino board
  • Passive buzzer
  • Jumper wires
  • Breadboard

Wiring Instructions

  1. Connect the negative (‚Äì) pin of the buzzer to GND.
  2. Connect the positive (+) pin to pin 3 (or any ~ pin).

Code Overview

1. Play a Note

The tone(pin, frequency, duration) function plays a note on the given pin. Try this inside setup():

tone(3, 440, 500);  // A4 note (440 Hz) for 500ms

2. Turn it on and off on command

At times, you're going to want to have the buzzer turn on or off depending on something, so you won't know the total duration ahead of time. If this is the case, you can just write tone(pin, frequency) instead, and it will play that note forever until stopped by noTone(pin).

Practice this by using tone(pin, frequency), delay(duration), and noTone(pin) to manually control when the buzzer goes on or off. This will be very useful later!

Experiment!

Try different frequencies like 262 (C4), 330 (E4), 392 (G4), or even tone(3, random(200, 1000)); for fun noise!
