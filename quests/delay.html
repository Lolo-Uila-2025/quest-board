Quest: Understanding delay()

The delay() function pauses your program for a set amount of time (in milliseconds). In this quest, you’ll use it to control the timing of messages printed to the Serial Monitor.

What You’ll Learn

  • How delay() works
  • How to control time between repeated actions
  • Why timing is important in Arduino sketches

Instructions

  1. Open the Arduino IDE and create a new sketch.
  2. In setup(), begin serial communication: 
    Serial.begin(9600);
  3. In the loop() function, add: 
    
Serial.println("Waiting...");
delay(1000); // wait 1 second (1000 milliseconds)
  4. Upload your code and open the Serial Monitor. You should see "Waiting..." printed once every second.
  5. Try changing the delay to 500, 2000, or 100. What happens?

To finish the quest, show your teacher the Serial Monitor output using different delay() values. Then, answer this question: How many times would a message print per second if you used delay(50) inside an infinite loop? If your answer is correct and your demo works, you’ll earn +5 points!
