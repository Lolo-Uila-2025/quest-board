Challenge: Is It Even?

Your mission is to write a function that checks if a number is even.

What To Do

  1. Create a function that takes a number as input (parameter).
  2. Use the modulus operator (%) to check if it's divisible by 2.
  3. Return true if it's even, false if not.
  4. Call the function a few times in setup() and print the result using Serial.println().

Example Output

true
false
true

Show your working code and Serial Monitor to the teacher to earn +5 points!
