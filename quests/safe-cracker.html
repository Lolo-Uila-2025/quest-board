Challenge: Safe Cracker

Turn a potentiometer to the secret code (a specific number) to “unlock” the safe and light an LED.

Required Components

  • Arduino board
  • Potentiometer
  • LED + 220 Ω resistor

Wiring

  1. Potentiometer middle pin → A0
  2. LED anode → pin 9 through resistor
  3. Give the potentiometer both positive and negative power at the bottom, and negative to the LED's other leg.

Logic Hints

  • Map analogRead(A0) (0‑1023) to 0–99 for a 2‑digit code.
  • Pick a random code at startup: secret = random(0,100).
  • If the user’s mapped value matches secret ±1, light up the LED for success.

🎯 Goal

Show you can “crack” the safe by dialing the potentiometer until the LED lights. Earn +15 points!
