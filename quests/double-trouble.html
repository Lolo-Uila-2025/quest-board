Challenge Quest: Double Trouble

This is a challenge quest — you're given the goal, but not the steps. Your mission: create a program that starts with a number and doubles it every second, printing each result. Stop after 5 times.

Goal

Your program should output something like this to the Serial Monitor:

2
4
8
16
32

Rules

  • Start with a number (like 2).
  • Every second, multiply it by 2 and print the result.
  • Repeat this exactly 5 times, then stop printing.
  • You may do this challenge with either a while or for loop in setup() or in loop().

Tips

  • Use a variable to store the current number.
  • Use delay() to wait 1 second between prints.
  • You may need a loop or a counter to know when to stop.
  • Don't forget to use Serial.begin(9600); in setup().

Show your completed program and Serial Monitor output to the teacher to earn +10 challenge points!
