Challenge Quest: Power Up!

In this challenge, you'll use a loop to print a number that **doubles every time** until it reaches or passes 100.

Goal

Your Serial Monitor should print something like this:

1
2
4
8
16
32
64
128

Rules

  • All code must be written inside the setup() function. Don't write inside the loop() function!
  • Start with the number 1.
  • Each time through the loop, multiply it by 2 and print it.
  • Keep looping until the number is greater than or equal to 100.

Tips

  • Use a variable to store the number.
  • You can use either a while loop or a for loop — your choice!
  • Don’t forget Serial.begin(9600); in setup().

Show your output to the teacher to earn +10 challenge points!
