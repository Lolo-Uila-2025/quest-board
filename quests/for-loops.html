Quest: Learning For Loops

A for loop is like setting a timer that knows three things:

  1. Where to start (the first value)
  2. When to stop (the condition)
  3. How to count (what changes every turn)

Real-Life Analogy

Picture climbing a five-step staircase:

  • Start on step 1
  • Keep climbing until you’re above step 5
  • Move up one step at a time

That’s exactly what a for loop does with numbers!

What You’ll Learn

  • The three parts of for(initial; condition; change)
  • How to repeat code a set number of times
  • How a for loop automatically stops when the condition is false

Your Task

Write a program that prints the numbers 1 through 5 using a for loop.

Instructions

  1. Create a new sketch in the Arduino IDE.
  2. All code goes in setup() (leave loop() empty).
  3. Start serial communication:
    Serial.begin(9600);
  4. Build your for loop. Remember its three parts: 
    // for (start ; keep-going? ; each-turn)
for (int i = 1; i <= 5; i++) {
  Serial.println(i);
}
    • int i = 1 → start at 1
    • i <= 5 → keep going while i is 5 or less
    • i++ → add 1 each turn

Hint: Change any of those three parts and watch how the loop behaves!

Expected Output

1
2
3
4
5

Show your Serial Monitor output to the teacher to earn +5 points.
