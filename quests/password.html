Challenge: Create a Password

In this challenge, you'll create a simple "password system" using buttons. The LED should only light up when a secret combination of button presses is entered correctly.

Required Components

  • Arduino board
  • 3 push buttons
  • 1 LED

  • Breadboard & jumper wires

Wiring Instructions

  1. Connect each button between an Arduino digital pin and GND.
  2. Use pinMode(pin, INPUT_PULLUP) to read each button.

How It Works

Your program should only turn the LED on if the correct combination is pressed. For example, you could require that:

  • Button 1 is not pressed
  • Button 2 is pressed
  • Button 3 is pressed

You can use digitalRead() to check the state of each button, and digitalWrite() to control the LED.
