Challenge Quest: Running Total

This challenge will test your ability to use a loop to keep track of a growing total.

Goal

Add up the numbers from 1 to 10, then print the total.

Your Serial Monitor should say:

The total is: 55

Rules

  • All code must be written inside the setup() function. Don't write inside the loop() function!
  • Use a loop to go from 1 to 10.
  • Create a variable that keeps track of the total.
  • After the loop finishes, print the result using Serial.println().

Tips

  • Start your total at 0.
  • Each time through the loop, add the current number to the total.
  • You can use total += i;

Show your working output to the teacher to earn +10 challenge points.
