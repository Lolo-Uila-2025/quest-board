Quest: Learning If Statements

In this quest, you’ll learn how to use if statements to make decisions in your code. This is one of the most important tools in programming!

What You’ll Learn

  • How to write an if condition
  • How to compare values using ==, >, <, etc.
  • How to make your program do different things based on different conditions

Instructions

  1. Open the Arduino IDE and create a new sketch.
  2. Start serial communication in setup(): 
    Serial.begin(9600);
  3. In loop(), create a variable called score and set it to any number (like 85).
  4. Use an if statement to check if score is greater than or equal to 90. The syntax is: 
    if (condition) {
    // code to execute
}
    For example: 
    if (score < 5) {
    Serial.println("Ok!");
}
  5. Test your program by changing the score value and uploading again.

Show your Serial Monitor output for at least two different values to the teacher to earn +5 points.
