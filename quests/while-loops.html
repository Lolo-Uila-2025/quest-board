Quest: Learning While Loops

In programming, we often want something to repeat — but only **while** a certain condition is true. That’s where a while loop comes in. It checks a condition, and as long as the condition is true, it keeps running the code inside.

Real-Life Analogy

Imagine this: You’re pouring water into a cup.

  • Before each pour, you check: "Is the cup full?"
  • If the answer is no, you pour more water and check again.
  • You repeat this **while the cup is not full**.
  • Once it is full, you stop.

That’s a while loop: check first → do something → repeat if still true.

What You’ll Learn

  • How to use a while loop to repeat actions
  • How to avoid infinite loops by updating your variables
  • Why conditions matter in programming logic

Your Task

Create a program that prints numbers from 1 to 5 using a while loop.

Instructions

  1. Open the Arduino IDE and create a new sketch.
  2. All your code must go inside setup() (leave loop() empty).
  3. Start serial communication: 
    Serial.begin(9600);
  4. Use a variable to track the number you're printing. Let the while loop keep printing numbers **as long as your condition is true**.
  5. Be sure to update the variable inside the loop — otherwise the loop will never stop!
  6. The syntax is: 
    while (condition) {
        // code to execute if condition is true
        }

Hint: Think: "What number do I start with? What number should I stop at? How do I move from one to the next?"
