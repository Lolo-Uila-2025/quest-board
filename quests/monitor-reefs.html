Story Quest: Monitor the Reefs!

Use a DHT11 sensor to monitor temperature and humidity as if you were observing environmental conditions on a coral reef!

Required Components

  • Arduino board
  • Breadboard
  • Jumper wires
  • 3 female-to-male jumper wires
  • DHT11 sensor module

The DHT11 sensor module will require the use of a library, click here to download it!

Wiring Instructions

  1. Left pin of DHT11 ‚Üí pin 2
  2. Middle pin ‚Üí positive
  3. Right pin ‚Üí negative

Tip: Use a breadboard so that it's easier to hook up an LCD later.

How It Works

The DHT11 sensor combines a temperature and humidity sensor and sends digital values to your Arduino. It works by detecting how electrical resistance changes with water vapor levels inside the sensor‚Äîpretty cool!

Code Setup

Install the DHT_nonblocking library and set up your code like this:

1. Include the Library

#include <DHT_nonblocking.h>
#define DHT_SENSOR_TYPE DHT_TYPE_11

2. Assign Pins and Create Sensor Object

static const int DHT_SENSOR_PIN = 2;
DHT_nonblocking dht_sensor( DHT_SENSOR_PIN, DHT_SENSOR_TYPE );

3. Begin Serial

Serial.begin(9600);

4. Create the Measuring Function

static bool measure_environment(float *temperature, float *humidity) {
  static unsigned long measurement_timestamp = millis();
  if (millis() - measurement_timestamp > 3000ul) {
    if (dht_sensor.measure(temperature, humidity) == true) {
      measurement_timestamp = millis();
      return true;
    }
  }
  return false;
}

5. Use the Values

float temperature;
float humidity;

if (measure_environment(&temperature, &humidity)) {
  Serial.print("T = ");
  Serial.print(temperature, 1);
  Serial.print(" deg. C, H = ");
  Serial.print(humidity, 1);
  Serial.println("%");
}
