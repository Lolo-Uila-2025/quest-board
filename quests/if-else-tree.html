Quest: Mastering If-Else Trees

In this quest, you’ll learn how to use if, else if, and else statements to handle multiple conditions in your code. This is a powerful way to make your programs smarter!

What You’ll Learn

  • How to write an if-else tree
  • How to handle multiple conditions using else if
  • How to provide a default action using else

Instructions

  1. Open the Arduino IDE and create a new sketch.
  2. Start serial communication in setup(): 
    Serial.begin(9600);
  3. In loop(), create a variable called score and set it to any number (like 75).
  4. Write an if-else tree:
    • If score is greater than or equal to 90, print "Excellent!".
    • If score is between 70 and 89 (inclusive), print "Good job!".
    • If score is below 70, print "Keep trying!".
  5. Explanation: The if statement checks a condition, and if it's true, the code inside its block runs. The else if statement allows you to check additional conditions, and the else block runs if none of the previous conditions are true.
  6. The syntax is: 
    if (condition1) {
        // code to execute if condition1 is true
        } else if (condition2) {
        // code to execute if condition2 is true
        } else {
        // code to execute if none of the above conditions are true
        }
  7. Test your program by changing the score value and uploading again.

Show your Serial Monitor output for at least three different values to the teacher to earn +5 points.
