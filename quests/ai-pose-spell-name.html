Teachable Machine: Spell Your Name with Poses

In this quest, you’ll use the Pose Project option in Teachable Machine to train an AI that recognizes body poses shaped like letters. When you finish, you’ll be able to spell out your name by striking each pose in front of the webcam.

🔧 What You’ll Need

  • Computer with webcam
  • Access to Teachable Machine

Step‑by‑Step Instructions

  1. Go to teachablemachine.withgoogle.com.
  2. Select Pose Project → Standard Pose Model.
  3. Create one class for each letter in your first name.
    (Example: for “CODY” make C, O, D, Y.)
  4. Record ≈ 30 pose samples per letter. Tip: exaggerate the letter shape so the model sees clear differences.
  5. Add No Pose class (optional) to help the model ignore random movements.
  6. Click “Train Model” and wait for it to finish.
  7. Use the live preview to test: strike each letter pose and watch the prediction bar.

🎯 Goal

Spell your entire name in order, on camera, with the model correctly identifying each pose. Earn +10 points!

💡 Tips

  • Stand in good lighting and keep the camera steady.
  • If two letters look similar, try adding accessories (e.g., raise one hand higher).
  • Retrain if the preview is confused or mixes letters.
