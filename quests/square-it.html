Challenge: Square It!

Can your function do math and return the result? Let’s try squaring a number!

What To Do

  1. Write a function named square() that takes one number.
  2. The function should return that number multiplied by itself.
  3. Call the function a few times and print the result using Serial.println().

Example Output

4
16
100

Show your working Serial output to the teacher to earn +5 points!
