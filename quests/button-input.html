Quest: Using a Button as Input

Required Components

  • 1 x Arduino board
  • 1 x Push button
  • Jumper wires

Wiring Instructions

  1. Connect one leg of the button to GND.
  2. Connect the other leg to pin 2 on the Arduino.
  3. No resistor needed! We‚Äôll use INPUT_PULLUP in code.

Code Setup

The goal of this program is to print a message every time the button is pressed.

1. Start Serial Communication

Inside setup(), begin the Serial Monitor so you can print messages to your screen:

Serial.begin(9600);

2. Configure the Button Pin

Set pin 2 to use INPUT_PULLUP mode:

pinMode(2, INPUT_PULLUP);

This keeps the pin HIGH by default. When the button is pressed, it gets pulled LOW, and we‚Äôll check for that.

3. Detect the Press in loop()

Read the pin using:

If state equals LOW, it means the button is pressed. Use an if-statement to print a message:

if (state == LOW) {
  Serial.println("Button Pressed!");
}

4. Add a Delay

Finally, slow it down a bit so it doesn‚Äôt spam the Serial Monitor:

delay(100);

Remember: When the button is pressed, the circuit is completed and pin 2 reads LOW!
