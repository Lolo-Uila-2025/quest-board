Story Quest: Menu Navigation Interface

In this quest, you‚Äôll create a text-based interface using the Serial Monitor and buttons. The goal is to switch between menus using just buttons, kind of like navigating a game or vending machine!

Required Components

  • Arduino board
  • 2 push buttons

Wiring Instructions

  1. Connect one leg of each button to GND.
  2. Connect the other leg of button 1 to pin 4.
  3. Connect the other leg of button 2 to pin 3.

Code Structure

You‚Äôll write separate void functions for each menu. Your program will print text using Serial.println() and wait for button input before jumping to another function.

Wait‚ÄëHelper Functions

void waitForPress() {
  // pause here until EITHER button reads LOW (pressed)
  while (digitalRead(button1) == HIGH && digitalRead(button2) == HIGH) {
    // just wait
  }
}

void waitForRelease() {
  // keep waiting until BOTH buttons go back to HIGH (released)
  while (digitalRead(button1) == LOW || digitalRead(button2) == LOW) {
    delay(10);   // tiny pause to avoid bouncy presses
  }
}

Super‚Äësimple explanation: waitForPress() freezes the program until someone presses a button. waitForRelease() makes sure the button is let go before moving on. This stops accidental double‚Äëpresses.
