Teachable Machine: Hand Signs

In this quest, you'll train a simple image classification AI using your webcam! You'll teach it to recognize common hand gestures and use that model to predict them in real time.

🔧 What You’ll Need

  • Computer with webcam
  • Access to Teachable Machine

🔧 Step-by-Step Instructions

  1. Go to teachablemachine.withgoogle.com
  2. Click "Get Started" and choose Image Project → Standard Image Model
  3. Create the following classes (you’ll need at least 6):
    • Rock
    • Paper
    • Scissors
    • Shaka
    • Peace Sign
    • No Hands
  4. Record webcam samples for each class (at least 30 samples per class is a good goal)
  5. Click "Train Model" and wait for it to finish
  6. Test your model using the webcam preview

🎯 Goal

Show your trained model in action—demonstrate it correctly identifies each of the six hand signs in real time. Earn +10 points!
