Story Quest: LCD Display

In this quest, you‚Äôll connect a 16x2 LCD screen to your Arduino and display messages using code.

Required Components

  • Arduino board
  • 16x2 LCD display (with HD44780 chip)
  • Potentiometer (for brightness)
  • Breadboard & jumper wires

Wiring Instructions

Use these pin connections for the LCD:

  • RS ‚Üí pin 7
  • E ‚Üí pin 8
  • D4 ‚Üí A0
  • D5 ‚Üí A1
  • D6 ‚Üí A2
  • D7 ‚Üí A3

The LCD needs a potentiometer connected to adjust the screen contrast: connect the middle leg of the potentiometer to V0 on the LCD, one leg to GND and the other to 5V.

Why These Pins?

You can use almost any digital or analog-capable pins for LCD connections. The only thing that matters is that the numbers you use in your code match the wiring you choose. For this quest, we picked pins 7, 8, and A0‚ÄìA3 for clarity and space.

Code Setup

You‚Äôll need to include the LiquidCrystal library to control the screen.

üîó If it's missing, you can download the LiquidCrystal library here (ZIP). Then go to Arduino IDE ‚Üí Sketch ‚Üí Include Library ‚Üí Add .ZIP Library to install it.

1. Include the Library and Setup Pins


#include <LiquidCrystal.h>

// RS, E, D4, D5, D6, D7
LiquidCrystal lcd(7, 8, A0, A1, A2, A3);

2. Start the LCD


void setup() {
  lcd.begin(16, 2);  // 16 characters, 2 lines
  lcd.print("Hello, world!");
}

3. Clear the Screen


lcd.clear();  // Clears everything

4. Move to a Specific Line


lcd.setCursor(0, 1);  // column 0, row 1
lcd.print("Line 2 text");
