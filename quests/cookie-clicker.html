Challenge: Cookie Clicker

Build a mini “Cookie Clicker” game! Button 1 earns cookies; Button 2 buys an upgrade so each future click earns more cookies.

Required Components

  • Arduino board
  • 2 push buttons
  • LCD display or Serial Monitor

Wiring

  1. Button 1 → pin 2 + GND
  2. Button 2 → pin 3 + GND

Game Logic

  • Start with int cookies = 0 and int valuePerClick = 1.
  • When button 1 pressed, add valuePerClick to cookies.
  • Upgrade cost = 10 cookies × current level. If Button 2 pressed and you can afford, subtract cost and add 1 to valuePerClick.
  • When testing, holding down the button might count as hundreds of clicks a second. To stop this, add code to pause the program until the button is released.
  • while (digitalRead(2) == LOW);

Use lcd.print() or Serial.print() to show cookies and upgrade cost.
