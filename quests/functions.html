Quest: Writing Your Own Functions

In this quest, you'll learn how to make your own function — a way to organize your code so you can use the same block of instructions over and over again.

What Is a Function?

A function is like a recipe. You give it a name and a list of steps, and you can “call” it whenever you want to follow those steps.

Example in real life:

  • You create a recipe called makeToast()
  • Inside the recipe: put bread in toaster, push button, wait
  • Anytime you want toast, you just call makeToast()

Your Task

You’re going to write your own function that prints a message to the Serial Monitor. Then you’ll call it from the setup() function.

Instructions

  1. Open the Arduino IDE and create a new sketch.
  2. Inside setup(), start the Serial Monitor: 
    Serial.begin(9600);
  3. Write your own function above setup(). Give it the name greet() and make it print a greeting: 
    
void greet() {
  Serial.println("Hello, I am a custom function!");
}
  4. Inside setup(), call your function by name: 
    greet();
  5. Upload your sketch and open the Serial Monitor. You should see your greeting message appear.

Great job! You just made a reusable block of code. Show your output to the teacher to earn +5 points!
