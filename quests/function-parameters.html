Quest: Function Parameters

You’ve already learned how to make your own function. In this quest, you’ll take it a step further: your function will be able to accept input — called a parameter.

What’s a Parameter?

Think of a parameter like an ingredient you pass into a recipe.

  • Imagine a function named makeSandwich().
  • What if you want to choose the filling (like ham, turkey, or peanut butter)?
  • You can add a parameter like this: makeSandwich("turkey");

Now the function can use that info to change what it does!

Your Task

Create a function that takes a number and prints: "The number is: [your number]"

Instructions

  1. Open the Arduino IDE and create a new sketch.
  2. Start with the basic setup code: 
    void setup() {
  Serial.begin(9600);
}
void loop() {
}
  3. Above setup(), create your own function like this: 
    
void printNumber(int num) {
  Serial.print("The number is: ");
  Serial.println(num);
}
  4. Then inside setup(), try calling your function with different values: 
    
printNumber(42);
printNumber(7);
printNumber(123);
  5. Upload your code and open the Serial Monitor. You should see the results!

Expected Output

The number is: 42
The number is: 7
The number is: 123

Once it works, show your Serial Monitor to the teacher to earn +5 points!
