Quest: Returning Values from Functions

Last time, you made a function that took in a value (a parameter). Now it’s time to learn how your function can send a value back. That’s called a return.

What Does return Mean?

Imagine a vending machine: you press a button, and it gives you something back. That’s what a function with a return does — you call it, and it sends a value back to you.

For example:


int doubleIt(int num) {
  return num * 2;
}

When you call doubleIt(5), it gives you back 10.

Your Task

Create a function that takes a number, doubles it, and returns the result. Then print that result to the Serial Monitor.

Instructions

  1. Open the Arduino IDE and create a new sketch.
  2. Start your setup() and loop(): 
    
void setup() {
  Serial.begin(9600);
}

void loop() {
}
  3. Above setup(), write a function that takes an int, doubles it, and returns the answer: 
    
int doubleIt(int num) {
  return num * 2;
}
  4. Inside setup(), call the function and print the result: 
    
int result = doubleIt(6);
Serial.println(result);
  5. Try it again with a different number!

Expected Output

12

Show your Serial Monitor output and working code to the teacher to earn +5 points!
