Challenge: The Bigger One

Write a function that compares two numbers and tells you which one is bigger.

What To Do

  1. Create a function called biggerOf() that takes two numbers.
  2. Inside the function, use an if statement to compare them.
  3. Return the bigger number.
  4. In setup(), call the function with different numbers and print the result.

Example Output

5
12
99

Show your working code and Serial Monitor output to the teacher for +5 points.
