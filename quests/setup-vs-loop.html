Quest: Setup vs Loop

This quest challenges you to demonstrate your understanding of how an Arduino sketch runs. You’ll explain the difference between setup() and loop() — two of the most important functions in the Arduino IDE.

What You’ll Learn

  • What setup() is used for
  • What loop() is used for
  • Why the Arduino needs both

Instructions

  1. Open the Arduino IDE and look at the default code: 
    void setup() {
  // runs once
}

void loop() {
  // runs over and over
}
  2. Think about how the Arduino behaves when you upload a sketch. What happens first? What keeps repeating?
  3. Prepare a short explanation of:
    • What setup() does
    • What loop() does
    • One example of something that might go in each function
  4. Explain your answer to a Kumu when you’re ready!

If your explanation is clear and correct, you’ll earn +5 points for this quest!
