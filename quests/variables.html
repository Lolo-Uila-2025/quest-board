Quest: Variable Basics

The goal is to:

  • Declare variables of different types
  • Assign values to them
  • Print those values to the Serial Monitor

Instructions

  1. Open the Arduino IDE and create a new sketch.
  2. Declare three variables:
    • An int named score set to 100
    • A float named accuracy set to 99.5
    • A bool named isGameOver set to false
  3. Use Serial.println() to print out each variable's value, like this: 
    Serial.println(score);
  4. Upload your sketch to the Arduino and open the Serial Monitor (magnifying glass icon) to view your printed variables.

When you're done, show your output to the teacher. If everything looks good, you'll earn +5 points!
