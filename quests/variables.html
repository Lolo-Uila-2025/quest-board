Quest: Variable Basics

In this quest, you'll learn how to store information in your code using something called variables.

What Is a Variable?

A variable is like a labeled box in your code. You can put a value inside it — like a number, a score, or a true/false — and your program can read or change what's inside later.

For example:

  • score could be 100
  • accuracy could be 99.5
  • isGameOver could be false (meaning the game is still going)

Your Task

You're going to write a program that creates three variables, gives them values, and prints those values to the Serial Monitor.

Instructions

  1. Open the Arduino IDE and create a new sketch.
  2. In the setup() function, start your serial monitor: 
    Serial.begin(9600);
  3. Right after that, create three variables with these values:
    • An int named score set to 100: 
      int score = 100;
    • A float named accuracy set to 99.5
    • A bool named isGameOver set to false
  4. Use Serial.println() to print out each of your variables: 
    
  Serial.println(score);
  Serial.println(accuracy);
  Serial.println(isGameOver);
  5. Upload your code and open the Serial Monitor (magnifying glass icon in the top right) to see your results.

When you're done, you should see all three values printed in the Serial Monitor window. Show your teacher to earn +5 points!
