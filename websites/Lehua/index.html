Topic 1: What is an Arduino?

An Arduino is a circuit board that you can connect to your computer so it can do different things (ex: light up LEDs, set off buzzers, etc).

Topic 3: Favorite Arduino part?

My favorite Arduino part has got to be the LCD because you can write things on it and it can stay there until you change it!

Topic 5: Why is encryption and decryption important? And an example of a Caeser Cipher.

Encryption and decryption is important because it ensures that our data, passwords, everything, is kept safe and protected from hackers. An example of Caeser Cipher would be: Ef! Qefp fp xk buxjmib! This can be decrypted to: Hi! This is an example!
